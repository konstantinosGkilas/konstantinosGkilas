## About

Software engineer with 5 years of experience building production backend systems, open source libraries, and distributed infrastructure. I gravitate toward problems where correctness, performance, and scale intersect — graph data structures, distributed coordination, and observable systems.

Primary language is Java, with a growing focus on Rust for systems-level tooling. I believe good engineering is boring in the best way: predictable, well-observed, and easy to reason about.
